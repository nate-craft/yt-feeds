## Installing

`cargo install --git https://github.com/higgsbi/yt-feeds`

YT-Feeds requires mpv and yt-dlp to work at runtime
- [mpv](https://github.com/mpv-player/mpv)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
